Mi Ruta de Aprendizaje 2022-2023

HTML:

Lenguaje de marcado utilizado para crear páginas web. Define la estructura y contenido de una página web.

Nivel: Medio
CSS:

Lenguaje utilizado para dar estilo y presentación a las páginas web.

Nivel: Medio
JavaScript:

Lenguaje de programación que se utiliza principalmente para crear interactividad en una página web, ya sea mediante la creación de animaciones, validaciones de formularios, entre otras cosas.
Java:

Lenguaje de programación orientado a objetos y de propósito general que se utiliza en una amplia variedad de aplicaciones, desde desarrollo de software hasta desarrollo de aplicaciones móviles y web.
MySQL:

Sistema de gestión de bases de datos relacionales de código abierto que se utiliza para almacenar y gestionar grandes cantidades de información de manera eficiente y segura.
